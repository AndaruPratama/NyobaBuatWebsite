KEGIATAN SAYA

Saya merupakan anak magang dari HMSI (Himpunan Mahasiswa Sistem Informasi) divisi Socdev atau Social Development. Disini saya banyak mendapat kenalan baru serta dapat mengetahui seluk beluk kegiatan divisi Social Development beserta program kerjanya.

Lalu, saya juga mengikuti sebuah acara bernama SIMETRI yaitu Sistem Informasi Mengajar Teknologi Informasi. Saya sebagai volunteer pengajar, bertugas untuk menyampaikan sebuah materi seputar html, css, serta bootstrap kepada adik-adik SMA/SMK domisili Surabaya.
